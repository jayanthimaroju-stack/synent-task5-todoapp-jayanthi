# To-Do List Web Application

## 📌 Project Overview

The To-Do List Web Application is a simple and interactive task management system developed using HTML, CSS, and JavaScript. It allows users to create, organize, and manage their daily tasks efficiently. The application stores tasks using the browser's Local Storage, ensuring that tasks remain available even after refreshing or reopening the browser.

---

## 🎯 Objectives

* Develop a responsive task management application.
* Practice JavaScript DOM manipulation.
* Implement Local Storage for persistent data.
* Improve user experience with an interactive interface.

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

---

## ✨ Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Filter tasks (All, Completed, Pending)
* Automatically saves tasks using Local Storage
* Responsive and user-friendly interface
